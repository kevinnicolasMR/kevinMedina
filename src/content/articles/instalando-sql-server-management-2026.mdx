## Instalación paso a paso

### 1. Ejecutar el instalador

1. Abre el archivo `.exe` descargado.
2. Acepta los términos de licencia.
3. Elige la ruta de instalación si quieres personalizarla.

### 2. Completar la instalación

1. Haz clic en **Instalar**.
2. Espera a que finalice el proceso.
3. Reinicia el equipo si el instalador lo solicita.

### 3. Abrir SSMS por primera vez

1. Busca **Microsoft SQL Server Management Studio** en el menú Inicio.
2. Conéctate a una instancia local o remota de SQL Server.
3. Crea una consulta de prueba:

```sql
SELECT @@VERSION AS version_sql_server;
```

## Problemas frecuentes

### El instalador no inicia

- Ejecuta el instalador como administrador.
- Verifica que tu antivirus no bloquee la descarga.
