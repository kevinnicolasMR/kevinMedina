## Cómo instalar SQL Server Management Studio + SQL Server Express en Windows

Instalar **SQL Server Express** y **SQL Server Management Studio** puede parecer confuso al principio, sobre todo porque Microsoft separa el motor de base de datos de la herramienta visual que usamos para administrarlo.

En este tutorial vas a ver cómo instalar ambas cosas paso a paso:

- **SQL Server Express**, que es el motor de base de datos.
- **SQL Server Management Studio**, también conocido como **SSMS**, que es la interfaz visual para gestionar bases de datos, tablas, consultas y conexiones.
- La configuración inicial de la instancia.
- El inicio de sesión con autenticación de Windows o usuario `sa`.
- Qué revisar si SQL Server no conecta o aparece como inaccesible.

La idea es que puedas dejar SQL Server funcionando en tu computadora para practicar, estudiar, hacer consultas, crear bases de datos o trabajar con proyectos locales.

---

## Qué diferencia hay entre SQL Server Express y SQL Server Management Studio

Antes de instalar nada, conviene entender una diferencia clave.

**SQL Server Express** es el motor. Es decir, es el servicio que permite crear y administrar bases de datos relacionales en tu computadora.

**SQL Server Management Studio**, o **SSMS**, es la herramienta visual que usamos para conectarnos a ese motor y trabajar de una forma más cómoda.

Sin SSMS también podrías manejar SQL Server desde consola, pero para la mayoría de casos no tiene mucho sentido complicarse. Hoy lo normal es usar una interfaz visual para crear tablas, escribir queries, hacer backups, revisar datos y gestionar instancias.

Dicho de forma simple:

| Herramienta | Para qué sirve |
|---|---|
| SQL Server Express | Es el motor de base de datos |
| SQL Server Management Studio | Es la interfaz visual para administrar SQL Server |

Por eso en este tutorial vamos a instalar primero el motor y después la herramienta de administración.
